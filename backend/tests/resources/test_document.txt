Personal Agent Project Documentation

This is a test document for the Personal Agent MVP project. The Personal Agent is an AI-powered assistant that can help with various tasks including:

1. Mathematical Calculations
   - Basic arithmetic operations
   - Complex mathematical expressions
   - Statistical calculations

2. Time and Date Information
   - Current time and date
   - Timezone conversions
   - Calendar information

3. Document Q&A (NEW FEATURE)
   - Upload PDF documents
   - Ask questions about document content
   - Search through uploaded documents
   - Get relevant excerpts and answers

Key Features:
- RAG (Retrieval-Augmented Generation) for document processing
- Vector embeddings using OpenAI's text-embedding-ada-002
- Semantic search with cosine similarity
- Document chunking for better retrieval
- User-specific document storage

Technical Stack:
- Backend: FastAPI with Python
- Database: SQLite with SQLAlchemy
- Vector Storage: FAISS
- Document Processing: PyPDF2
- Text Splitting: LangChain
- Embeddings: OpenAI
- Frontend: HTML/CSS/JavaScript

This document serves as a test case for the document Q&A functionality. Users should be able to upload this document and ask questions like:
- "What are the key features of the Personal Agent?"
- "What technical stack is used?"
- "How does document Q&A work?"
- "What type of calculations can the agent perform?"

The system should be able to retrieve relevant information and provide accurate answers based on the document content.
