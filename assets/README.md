## Creating Assets

### For Screenshots
1. Start the application (`./start_server.sh`)
2. Navigate to http://localhost:8000
3. Take high-quality screenshots of key features
4. Use macOS Screenshot.app (⌘⇧5) or similar tools

### For GIFs
1. Use screen recording tools:
   - **macOS**: Screenshot.app (⌘⇧5) 
   - **Cross-platform**: OBS Studio, LICEcap
2. Convert recordings to GIF using online tools or FFmpeg
3. Keep file sizes reasonable (<10MB) for GitHub

### Optimization
- Compress images to reasonable sizes
- Use WebP format for better compression when possible
- Ensure screenshots are readable (good contrast, appropriate size)
